# 당근마켓 클론 챌린지

## 구현 기준

- Spring Boot
- 모바일 웹 기준
- 데이터베이스를 설계하고 ERD 작성
- AWS EC2, RDS 활용
- JPA 활용

## 기능 요구사항

1. 회원가입/로그인 기능
2. 상품등록 기능
3. 상품페이지 기능
4. 마이 페이지(나의 당근 페이지) 기능
5. 1:1 채팅 기능 (웹 소켓통신 채팅 기능 구현)
